# X 帳號分析

有很多網站專門為你提供 X 帳號的定性分析。雖然 X 已經提供了一個**分析**板塊，但它更側重於展示你的資料表現。

而定性分析關注的是你貼文的品質，而不是表現統計。你可以從這類分析中取得的一些洞察包括：
- 什麼規律讓我的貼文爆紅？
- 我談論哪些話題能取得最多互動？
- 為什麼我有些貼文能取得 1000+ 個讚，但有時貼文只有不到 5 個讚？我做錯了什麼？

有很多網站和應用程式提供 X 分析，但它們側重於統計資料。可能只有 1-2 個網站允許你與 AI 對話來了解你的表現。

但現在你可以使用 OpenClaw 來為你做這個分析，無需在這些網站上支付每月 10-50 美元的訂閱費。

## 所需技能
Bird 技能。`clawhub install bird`（已預裝）

## 如何設定
以下是流程：
1. 確保 Bird 技能正常運作。
2. 為了安全和隔離，最好為你的 ClawdBot 建立一個新帳號。
3. 使用你的 X 帳號進行認證。在 Chrome/Brave 中登入 x.com，並提供正確的 cookie（網頁憑證）資訊（`auth-token`、`ct0`），以便 OpenClaw 可以存取你的帳號。
4. 讓 OpenClaw 檢視你的真實帳號，取得最近 N 條推文，然後你可以隨意提問。或者，你也可以讓它為你編寫特定的腳本。
